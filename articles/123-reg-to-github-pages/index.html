Pointing your domain to github pages

After building this blog statically and pushing it up to github as a project page, I then wanted to point my domain at it.

  1. In the root of my website create a file called CNAME and put the name of my domain in it spiraltechnology.co.uk. To do this in a wintersmith generated site, you need to drop the file into the contents directory before generating the site. This is then generated by wintersmith and pushed to the gh branch of the website
cd ~/dev/spiraltechnology-website
wintersmith build
cd ~/dev/spiraltechnology-website-deploy
git add CNAME
git commit -a -m "Adding the CNAME for custom domain"
git push origin gh-pages
